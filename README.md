# wkhtmltopdf
Install wkhtmltopdf and wkhtmltoimage with NPM or YARN on Linux/OSX Distro

# Installation
```shell
npm install bin-wkhtmltopdf -g
# OR
yarn add bin-wkhtmltopdf -g
```
# Usage
```shell
wkhtmltox-install
# If error try with Sudo
sudo wkhtmltox-install
```
